Character Counter

A simple **JavaScript Character Counter** web application that allows users to type a message in a textarea and see the number of characters used in real-time. The counter updates dynamically and prevents users from exceeding the maximum character limit.

Features

1. Real-time character counting while typing
2. Displays typed characters out of the maximum limit
3. Prevents typing beyond **200 characters**
4. Shows a warning message when the limit is reached
5. Clean and simple user interface

 Technologies Used

1. **HTML5** – Structure of the webpage
2. **CSS3** – Styling and layout
3. **JavaScript (DOM Manipulation)** – Real-time character counting logic

How It Works

1. The user types a message inside the textarea.
2. JavaScript listens for the `input` event.
3. The application counts the number of characters typed.
4. The counter updates automatically (example: `45/200 characters`).
5. If the user reaches the **200-character limit**, a warning message appears and further typing is prevented.
